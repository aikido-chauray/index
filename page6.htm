Comment intégrer une carte sur google map de votre tablette ou Smartphone, Exemple :
1. Télécharger le fichier ci-après Dojos de l'Ouest
2. Connectez-vous sur votre compte Google sur un ordinateur
3. Ouvrez Google map puis allez sur "My Map"
4. Créer une carte puis "Importer"
5. sélectionner votre fichier télécharger "Dojos de l'Ouest.kml" puis Ok
